# 💬 Mehfil - Real-Time Chat Application

**Mehfil** is a modern real-time chat application built using the **MERN Stack** and **Socket.IO**. It allows users to chat instantly with a clean and responsive interface inspired by modern messaging apps.

---

## ✨ Features

- 🔐 User Authentication (Login / Signup)
- 💬 Real-Time One-to-One Chat
- ⚡ Instant Messaging with Socket.IO
- 🟢 Online / Offline User Status
- ⌨️ Typing Indicator
- 🔔 Notification Sound
- 📱 Fully Responsive Design
- 🎨 Clean Dark UI
- ⚛️ Redux Toolkit State Management

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- DaisyUI
- Redux Toolkit
- Axios
- Socket.IO Client

### Backend
- Node.js
- Express.js

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/Pratik-Sarse/mehfil-chat-app.git
```

### Client

```bash
cd Client
npm install
npm run dev
```

### Server

```bash
cd Server
npm install
npm start
```

---

## 🔑 Environment Variables

Create a `.env` file inside the **Server** folder.

```env
PORT=5000
MONGO_URI=YOUR_MONGODB_CONNECTION
JWT_SECRET=YOUR_SECRET_KEY
CLIENT_URL=http://localhost:5173
```

Create a `.env` file inside the **Client** folder.

```env
VITE_DB_URL=http://localhost:5000/api/v1
VITE_DB_ORIGIN=http://localhost:5000
```
